## 它能做什么

Aristotle 不是一个思想家——是 **2300 年来 4 大解读传统的对象**(经院 Aquinas / 阿拉伯 Avicenna / 海德格尔派 / 后分析 MacIntyre)。本镜片是 v0.5 traditional 学者**首个验证案例**:

> 「**亚里士多德怎么看 X**」

→ **默认 multi-perspective 输出**:本人 + 4 派各自看,共 5 段回答,每段附该派代表作引文锚点。

> 「**用 Aquinas / 经院派看 X**」/「**用 Heidegger 派的 Aristotle reading 看 X**」

→ 单派视角输出。

## Lineages · 4 派传统

| Lineage | 代表 | 活跃期 | 核心 Reading Strategy |
|---|---|---|---|
| **1. 经院 / Thomism** | Thomas Aquinas | 13c | 把 Aristotle 整合进基督教神学(essentia/esse + acquired/infused virtue 二分) |
| **2. 阿拉伯 / Falsafa** | Avicenna (Ibn Sina) | 10-11c | Aristotle + 新柏拉图 + 一神论合成,首创 essence/existence 区分 |
| **3. 海德格尔 / Phenomenology** | Martin Heidegger | 20c | 回到 *physis / aletheia / ousia* 原始意义,反 metaphysica 化 |
| **4. 后分析 / Neo-Aristotelian** | Alasdair MacIntyre | 20-21c | 用分析哲学复兴德性论(narrative + practice + tradition 三层结构) |

### 4 派次级影响线(参考)

- 新柏拉图派(Plotinus / Porphyry)
- 犹太派(Maimonides)
- Averroes / 拉丁 Averroism
- Suárez 与晚期经院

## 诚实边界

本镜片**不是 Aristotle 本人,也不是任一派的"标准" reading**。

它是基于 Aristotle 著作 + 4 派代表学者著作的多视角分析工具,承载着:
- **默会知识丢失**——Aristotle 的研究品味不可蒸馏
- **公开 vs 私下分裂**——现存 *Corpus Aristotelicum* 是 *esoteric*(内部讲义),Aristotle 公开发表的 *exoteric* "对话"早已散失
- **2300 年传记修辞污染**——Diogenes Laërtius 距 Aristotle 600 年,所有生平叙述已严重叙事化
- **化石化**——著作闭合,但接受史仍在演化
- **漫画化风险**——警惕产出"Aristotle 口头禅集合"
- **死亡-尊重边界**——322 BC 死亡无戏剧化解读
- **lineage 投射边界**(v0.5)——4 派的 reading 是各派的"创造性误读"(Bloom 意义),**不是 Aristotle 本人意图**

详见 [`SKILL.md`](SKILL.md) 诚实边界章节。
