## 用户操作

1. 浏览器打开 annas-archive.org / Galilée 官网 / 学者主页 等
2. 找到对应书的 PDF 下载到 `~/Downloads/`
3. 跑 intake:
```bash
python3 scripts/intake_manual_pdf.py \
    ~/Downloads/{file}.pdf \
    --config examples/stiegler-perspective/references/research/_library_config.md \
    --manifest-id <从上表 .json 复制 id> \
    --execute
```
