## 它能做什么

把 Stiegler 的**概念地图 + 方法论进路**装到 Claude / 任何 LLM agent 里。读到一段技术现象材料(AI / 平台 / 算法 / 数字工业 / 政治经济),问:

> 「**用 Stiegler 视角看这个材料**」

→ 镜片自动调用 6 个核心概念(*pharmakon* / 第三持存 / 普遍器官学 / 后种系生成 / 象征贫困 / 三重废人化)+ 3 个次级概念(grammatization / disruption / neganthropy),按 Stiegler 实际写作中的论证节奏给出第三人称分析。

**v0.5.2 新增**:可触发**多 lineage 视角**——
> 「**用 Stiegler 海德格尔继承面看 X**」/「**Stiegler 与 Marx 在 X 上的对话**」/「**中国 Stiegler 派(许煜)看 Y**」

## 诚实边界

本镜片**不是 Stiegler 本人**。它是基于公开材料的分析工具,承载着:
- **默会知识丢失**——研究品味不可蒸馏
- **公开-私下过滤**——所有公开材料都是经过过滤的展演自我
- **传记修辞污染**——所有传记类信息已严格分级
- **化石化时间窗**——调研截止 2026-05-05 的快照
- **漫画化风险**——不要让本镜片变成"Stiegler 口头禅集合"
- **死亡-尊重边界**——Stiegler 2020 自杀事实公开,不戏剧化、不连接因果
- **派学者投射边界**(v0.5.2)——4 派 lineages 是塑造他思想的传统,不是他本人立场

详见 [`SKILL.md`](SKILL.md) 诚实边界章节。
